Safety

Is it safe?

Short answer: yes. Here is exactly why, and how to check for yourself.

  • Reads logs onlyWatches the VRChat process and its own local log files. It never injects code, modifies the game, or automates play, and it isn’t a mod or a cheat. See the privacy page for exactly what it reads and sends.
  • Built in the openEvery build is published as a GitHub Release, and the button above always points at the official latest-release file.
  • No admin neededA portable single .exe: no installer, no elevated permissions, and deleting the file uninstalls it.
  • Verify it yourselfScan with VirusTotal, or hash it in PowerShell with Get-FileHash .\AeroVRC.exe -Algorithm SHA256 and compare against the SHA-256 shown for the asset on the GitHub release.

The one warning you will see

AeroVRC isn’t code-signed (certificates are costly for a free tool), so Windows SmartScreen shows a generic “unrecognized app” prompt the first time you run it. Three clicks:

  1. Windows shows “Windows protected your PC.” Normal for any unsigned app.
  2. Click “More info”. The prompt expands to show the app’s name.
  3. Click “Run anyway”. That’s it; Windows only asks the first time.
